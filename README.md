# Estética Martins - Sistema de Agendamento

## Descrição

O projeto "Estética Martins" é um sistema de agendamento desenvolvido para uma clínica de estética. O sistema permite o gerenciamento de serviços, funcionários, clientes e agendamentos, oferecendo uma interface simples para agendar, confirmar e cancelar serviços.

## Tecnologias Utilizadas

- **Backend**: Node.js
- **Banco de Dados**: MySQL
- **Frontend**: HTML, CSS, JavaScript (opcional)
